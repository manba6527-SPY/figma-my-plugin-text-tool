# Figma 字号统一修改工具

一款用于快速识别、统计和统一修改 Figma 设计文件中文本字号的自动化工具。

## 功能特性

### 核心功能
- 📊 **自动扫描文本节点**：支持扫描当前页面或选中图层组内的所有文本节点
- 📈 **字号统计与可视化**：按字号大小排序展示使用次数
- 🔍 **源字号选择**：点击列表快速选择需修改的源字号
- 🎯 **目标字号设置**：支持手动输入目标字号（≥1px）
- ✨ **批量统一修改**：一键替换所有匹配的源字号为目标字号
- 🎨 **支持混合字号文本**：可精确修改同一文本框内的特定字号片段
- 📋 **修改结果反馈**：实时显示修改数量和更新后的字号分布

### 辅助功能
- 🔄 **重新扫描**：手动触发扫描，适配设计稿修改场景
- ⏳ **操作状态提示**：显示扫描中、未找到文本等状态

## 安装步骤

1. 打开 Figma 桌面端
2. 点击右上角的 **插件** 图标 → **管理插件**
3. 点击 **开发** → **导入插件**
4. 选择本插件文件夹中的 `manifest.json` 文件
5. 插件安装完成后，可通过 **插件** → **开发** → **字号统一修改工具** 启动

## 操作流程

### 基本操作（≤3步）
1. **启动插件**：插件自动扫描当前页面文本节点
2. **选择源字号**：在列表中点击需修改的字号
3. **输入目标字号**：在输入框中输入新的字号值
4. **点击修改**：自动将所有源字号替换为目标字号

### 高级操作
- **扫描选中图层**：先在 Figma 中选择图层组，再启动插件即可只扫描选中内容
- **重新扫描**：设计稿修改后，点击右上角的「重新扫描」按钮更新统计

## 界面说明

| 区域 | 功能描述 |
|------|---------|
| 标题栏 | 显示插件名称和重新扫描按钮 |
| 状态区 | 显示当前操作状态（扫描中、修改成功等） |
| 字号列表区 | 按大小排序显示所有字号及使用次数，点击选择源字号 |
| 输入区 | 显示选中的源字号（只读）和输入目标字号 |
| 操作按钮 | 统一修改按钮，完成前置操作后启用 |

## 技术特性

- **性能优化**：支持扫描 1000+ 文本节点，响应时间 ≤3 秒
- **兼容性**：支持 Figma 桌面端最新版本，兼容混合字号、自动布局文本
- **易用性**：界面简洁，操作流程清晰，关键按钮智能禁用

## 注意事项

1. 插件仅修改文本的字号属性，不会影响其他文本样式（字体、颜色、行高、字间距等）
2. 混合字号文本的修改会保留其他样式不变，仅修改匹配的字号片段
3. 建议在修改前备份设计文件，防止意外修改
4. 如遇到问题，可尝试重新扫描或重启插件
